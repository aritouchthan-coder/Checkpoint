# Checkpoint Scanner

ระบบสแกน QR / Barcode สำหรับใช้งานบน GitHub Pages

## วิธีใช้งาน

1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub
2. เข้า Settings
3. ไปที่ Pages
4. เลือก Branch: main
5. กด Save
6. เปิดลิงก์ GitHub Pages

## หมายเหตุ

ข้อมูล Logs จะถูกเก็บใน browser ด้วย localStorage

ถ้าเปลี่ยนเครื่องหรือเคลียร์ browser ข้อมูลจะหาย
